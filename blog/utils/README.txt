The script reads the export xml file of your wordpress blog and updates DJPlus.

Written to shift wordpress blog to DJPlus

1) Change conn variable to point to a database engine of your preference(Mysql) 

2) Point the file variable to your exported wordpress xml file

3) Point the database blog.db file to your own db

4) Modify the 'postid' assignment to the 'id' value of your last post.

Thats all!
Run python parse.py and sit back!
